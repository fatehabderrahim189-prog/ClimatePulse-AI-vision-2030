# 🌍 ClimatePulse AI

## 🧠 Project Overview
ClimatePulse AI is an intelligent system that analyzes the impact of climate change on workforce productivity using machine learning and real climate data.

---

## 🚀 Features
- 🌡️ Climate data analysis (NASA API)
- 🤖 Machine Learning prediction model (XGBoost)
- ⚖️ Risk evaluation system
- 📊 Productivity scoring
- 🌍 Smart city decision support

---

## 🛠️ Technologies Used
- Python
- Streamlit
- Pandas
- Scikit-learn
- XGBoost
- NASA POWER API

---

## 📊 How it works
1. Collect climate data from NASA
2. Train ML model on temperature, humidity, solar data
3. Predict productivity impact
4. Generate risk and policy recommendations

---

## 🎯 Purpose
To help governments and organizations understand climate impact and improve smart city planning.
